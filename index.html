ENS Subdomains
This is a quick tool for setting up subdomains under any ENS domain you own.
For example if you own the ENS name geeogi.eth then you can get sub.geeogi.eth for free (2mins + tx fees).
Enter the sub you'd like (no spaces/caps/non-standard characters):

Domain that you own:

Address you'd like to point sub.domain.eth to:

1) Set the subnode owner to your address to allow you to control it.
Click the button below then confirm TX with Metamask/Mist and wait for it to be mined before proceeding to the next step.
2) Set the subnode to use the Public Resolver contract.
3) Set the subnode to point to your desired address.
That's it. Wait for the final TX to be mined and then search your subdomain on etherscan.io to confirm your new address.
P.s. if you're seeing a transaction error message then it's probably because you don't own the domain you entered. You must own the domain and send the TX from the owner account. See this great guide for help in setting up a domain.
